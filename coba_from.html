Jika kita mengklik tombol "Kirim", maka data formulir akan dikirim ke halaman bernama"/coba_form.html"

Jika menggunakan target _self akan berada di tab yang sama, jika menggunakan _blank akan membuat tab baru

Jika menggunakan method get di url akan menampilkan '/coba_form.html?i_name=Arenta&i_belakang=Smith', jika menggunakan method akan postnantinya akan menampilkan url dengan format 'file:///D:/coba_form.html'
